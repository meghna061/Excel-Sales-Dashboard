# Excel Sales Dashboard
## Overview
An interactive sales dashboard built in Microsoft Excel to analyze sales, profit, and business performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Files
### 📊 Superstore_Sales_Dashboard.xlsx
Contains the complete interactive Excel dashboard along with:
- Dashboard sheet
- Pivot tables
- Charts
- Supporting analysis used to create the dashboard

### 📂 Superstore_Raw_Data.xlsx
Contains the original Superstore dataset used for data cleaning, analysis, and dashboard creation.
